# Base Hype Wall

## App Name

Base Hype Wall

## Tagline

Light up support

## Description

Post one short support line on Base, attach it to a topic, and turn it into a bright onchain wall signal.

Character count: 108

## Category

Social / support / onchain utility

## Short Pitch

Base Hype Wall gives communities a fast way to drop short bursts of support and turn them into a colorful onchain signal wall.

## Production Environment Variables

- `NEXT_PUBLIC_HYPE_WALL_CONTRACT_ADDRESS`: deployed hype wall contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
